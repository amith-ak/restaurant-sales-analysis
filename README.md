# Restaurant Sales Data Analysis

## Project Overview

This project analyzes restaurant customer spending behavior using Python and data analysis libraries.
The dataset contains 244 customer transaction records including bill amount, tip, visit day, time, and group size.

## Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Objectives

* Understand customer spending patterns
* Identify high revenue days
* Analyze distribution of customer bills
* Visualize business insights using charts

## Key Findings

* The average customer bill is approximately $20.
* Sunday has the highest average customer spending.
* Most customers fall within a moderate spending range rather than very high bills.

## Visualizations

The project includes histograms and bar charts showing customer bill distribution and daily spending comparison.

## Conclusion

The analysis demonstrates how basic data analytics can help businesses understand customer behavior and plan staffing and pricing strategies more effectively.
